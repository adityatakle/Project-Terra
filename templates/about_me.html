Info about the "PROJECT TERRA"

This project is based on CS50x PSET9 Finance. This project focusses on concept of "Paper Trading" where the user can learn about stock market by investing in it with virtual money to learn about the market.

With this project the user can:

  • make a paper trading account
  • Can buy and sell stocks in US STOCK EXCHANGE "NASDAQ"
  • Can buy and sell crypto coins.
  • Can keep track of trades made.
