Info about the "PROJECT TERRA"

This project named "PROJECT TERRA" signifies the building the solid foundation for learning more about computer science and build real world application solutions.

This project is based on CS50x PSET9 Finance. It focusses on concept of "Simulation Trading" where the user can learn about stock market by investing using virtual money to learn about the market.

With this project the user can:

  • make a secure simulation trading account
  • Can buy and sell stocks in US STOCK EXCHANGE "NASDAQ"
  • Can buy and sell equity stocks in INDIAN STOCK EXCHANGE (NSE)
  • Can buy and sell CRYPTO COINS.
  • Make personalised watchlist to keep track of stocks and cryto coins.
  • Can keep track of trades made.

Skills We gained during making this project:

  • Integrating real time API's in web app.
  • Writing clean code
  • Honed the skills of logic building.
  • Leaned about using AI tools to help in debugging the code or finding silly mistakes made after writing code.
